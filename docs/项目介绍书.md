# 实训作业提交与批改系统 — 项目介绍书

> 一个面向职业院校的轻量级作业管理平台，老师发布作业、学生在线提交、自动统计成绩。

---

## 一、这个系统解决什么问题

在职业学校（中职/高职）的实训教学中，作业管理普遍存在以下问题：

| 痛点 | 现状 | 本系统方案 |
|------|------|-----------|
| 作业收集乱 | 微信群里发文件、U盘拷、邮件发，老师一个个收 | 学生统一在网页提交，自动归档 |
| 批改效率低 | 老师逐个打开文件看、手工记分、算平均分 | 在线打分+评语，自动统计 |
| 成绩难追溯 | 分数记在纸上或Excel，期末找不到 | 数据库永久保存，随时导出 |
| 谁交谁没交不知道 | 对着名单一个个对 | 系统自动标注已交/未交 |
| 班级管理散 | 学生名单靠口头传达 | 老师建班、学生自助注册加入 |

**一句话概括**：把作业的"发布→提交→批改→统计→导出"全流程搬到网页上，老师学生各用一个浏览器就能完成。

        ┌─────┴─────┐

  · 注册+建班      · 注册+选班加入

## 三、核心功能

### 老师端

**① 注册与建班**
- 注册时自动创建班级，自己成为班主任
- 可创建多个班级
- 班级管理页面支持增删

**② 发布作业**
- 标题 + 详细说明 + 选择班级 + 截止日期
- 发布后该班所有学生可见

**③ 批改作业**
- 一目了然的提交状态表格：已批改 / 待批改 / 未交
- 点击学生弹出批改窗口，可查看文字说明、下载附件
- 打分（0-100）+ 文字评语

**④ 成绩统计**
- 平均分、最高分、最低分、及格率
- Chart.js 柱状图展示分数段分布
- 每个学生的详细成绩明细

**⑤ 导出 Excel**
- 一键下载 xlsx 文件
- 包含学号、姓名、是否提交、提交时间、分数、评语、状态

### 学生端

**① 注册与选班**
- 注册时从已有班级列表中选择加入
- 可以随时切换班级

**② 查看作业**
- 按截止日期排序，已过期作业红色标注
- 已提交/已批改状态清晰展示

**③ 提交作业**
- 文字说明框（写解题思路、遇到的问题）
- 文件上传（代码文件、截图、文档）
- 已提交的可以修改

**④ 查看反馈**
- 老师评分和评语直接显示在作业卡片上

## 四、技术栈

| 层级 | 技术 | 选型理由 |
|------|------|----------|
| 后端框架 | Flask (Python) | 轻量、易学、适合教学项目 |
| 数据库 | SQLite | 零配置、单文件、无需安装数据库服务 |
| 前端 | HTML + CSS + 原生JS | 无框架依赖，浏览器直接打开 |
| 图表 | Chart.js (CDN) | 一行引入，柱状图开箱即用 |
| Excel导出 | openpyxl | 纯 Python 生成 xlsx |
| 部署 | 单文件运行 | `python app.py` 即可启动 |

---

## 五、快速开始

```bash
# 1. 安装依赖
pip install flask openpyxl

# 2. 初始化数据库（首次运行）
python init_db.py

# 3. 启动服务
python app.py

# 4. 浏览器打开
# 学生/老师注册: http://127.0.0.1:5001/register
# 登录: http://127.0.0.1:5001/login
```

## 六、项目结构

```
zuoye_system/
├── app.py              # Flask 主程序，所有路由
├── models.py           # 数据库操作（SQLite）
├── init_db.py          # 建库 + 预设演示数据
├── templates/          # 7 个页面模板
│   ├── layout.html             # 统一页面框架（导航栏+样式）
│   ├── login.html              # 登录页
│   ├── register.html           # 注册页（老师/学生双模式）
│   ├── teacher_home.html       # 老师首页
│   ├── teacher_classes.html    # 班级管理
│   ├── create_assignment.html  # 发布作业
│   ├── grade_list.html         # 批改页面（核心）
│   ├── stats.html              # 成绩统计+图表
│   └── student_home.html       # 学生首页+提交页+切换班级
├── uploads/            # 学生上传的文件
└── zuoye.db            # SQLite 数据库文件
```
